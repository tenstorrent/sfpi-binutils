SFPI-BINUTILS
=============

This is a fork of GNU Binutils for Tenstorrent Tensix processors, as a
submodule of `https://github.com/tenstorrent/sfpi`.  It is
(naturally) under the same GPL license(s) as upstream Binutils.

**Please see `https://github.com/tenstorrent/sfpi` for bug
  reporting information. Do not try to file a report in this repo.**
